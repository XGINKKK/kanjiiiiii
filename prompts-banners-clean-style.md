## Especificações Técnicas Finais:

### Dimensões:
- **Resolução**: 1500x1000px (3:2)
- **Formato**: JPG otimizado ou PNG
- **Peso**: Máximo 300KB após compressão

### Paleta de Cores (use nos prompts):
```
Coral principal: #FF6B6B
Coral suave: #FA8072
Mint vibrante: #51CF66
Mint suave: #4ECDC4
Navy escuro: #1A365D
Navy médio: #2D3748
Beige claro: #F5E6D3
Beige muito claro: #FFF4E6
Branco puro: #FFFFFF
```

### Estilo Consistente em Todos:
✅ Fundo clean (branco/bege gradiente)
✅ Elementos gráficos minimalistas (linhas pontilhadas)
✅ Iluminação suave e uniforme
✅ Sem sombras duras
✅ Profissional mas acessível
✅ Espaço limpo para texto/CTA
✅ Cores da marca integradas sutilmente

1. **Gere no Midjourney v6** primeiro (melhor qualidade fotográfica)

   - Otimize para web (< 300KB)
